# 🏥 Health & Wellness Sales Analysis & Visualization

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a Health & Wellness retail dataset using Python. The objective is to analyze sales performance, customer purchasing behavior, payment methods, and product line performance through data cleaning, statistical analysis, and visualizations.

The project demonstrates essential data analytics skills including data preprocessing, outlier detection, exploratory data analysis, and business insight generation using Python.

---

# 🎯 Objectives

- Assess data quality by identifying missing values and duplicate records.
- Perform data preprocessing and validation.
- Detect outliers using the Interquartile Range (IQR) method.
- Analyze sales performance across different product lines.
- Examine payment method preferences.
- Identify sales trends over time.
- Visualize business performance using various chart types.
- Generate actionable business insights from the dataset.

---

# 📂 Dataset

The dataset contains transactional information from a Health & Wellness retail business.
